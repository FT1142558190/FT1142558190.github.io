    win 安装git 安装 node

    node-v14.21.3-x64.msi

    node在win集成了npm 安装

    在powershell

    npm install hexo-cli -g

    -g是全局安装

    D:\codenew\Hexo>hexo init

    hexo s 跑起来了

    可能有个小坑 要改成 UTF-8的格式

    模板竟然也不是utf

    hexo new post

    git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly

    git clone -b master https://github.com/yelog/hexo-theme-3-hexo.git themes/hexo-theme-3-hexo

    三、应用hexo主题
    1.应用主题

    用VScode打开Hexo 根目录（博客源文件的文件夹）下的 _config.yml，

    拉到代码最下面，把主题（默认是theme: landscape）修改 为butterfly， hexo-theme-3-hexo

    theme: butterfly

    Ctrl+S保存，最小化VScode。

    一、初始化博客下 _config.yml

    1.1 国际化

    language: zh-CN #支持 zh-CN、en
yaml复制代码

    1.2 关掉 hexo 自带的代码高亮

    主题内置了主题高亮，所以需要禁用 hexo 自带的高亮

    根据 hexo 版本, 禁用的方式发生了变化 禁用高亮

    below v7.0.0

    # _config.yaml
highlight:
    enable: false
yaml复制代码

    7.0.0+

    # _config.yml
syntax_highlighter:  # empty

    2.5 开启 关于页面

    1. 在 hexo 根目录执行以下，创建 关于 页面
      hexo new page "about"
bash复制代码
    2. 位置： source/aoubt/index.md ，根据需要进行编辑。
    3. 在主题中开启显示：修改主题根目录 _config.yml 中的 about 的 on 为 true，如下所示
      menu:
about:  # '关于' 按钮
 on: true # 是否显示
 url: /about  # 跳转链接
 type: 1 # 跳转类型 1：站内异步跳转 2：当前页面跳转 3：打开新的tab页
